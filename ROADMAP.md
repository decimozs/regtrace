# Roadmap

Planned features, ordered rough-by-priority.

## Short term

- **Provider cost tracking** — fill missing model pricing in judge cost table; add user-configurable price overrides
- **Offline evaluation** — run factuality (shallow) and format metrics without any LLM provider configured
- **SARIF output** — emit SARIF for static analysis tool ingestion

## Medium term

- **Homebrew / npm / apt distribution** — install regtrace via standard package managers
- **GitHub Actions setup action** — `decimozs/setup-regtrace` for faster CI integration
- **Documentation site search** — add full-text search to docs

## Long term

- **Regtrace Dashboard** — optional hosted viewer for run history (not required, CLI remains the primary interface)

---

*This roadmap is a living document. Priorities shift based on community feedback. Open an issue if something here matters to you.*
